Summary : 오픈채팅 > 메시지 전송 시, 글로벌 검색 결과에 노출되지 않는 현상

Priority : High (검색은 메시지 탐색의 핵심 경로이며, 오픈채팅 메시지가 검색에서 누락되면 사용자가 해당 정보를 찾지 못해 실제 데이터 유실처럼 인지될 수 있음)
Severity : Major (기능 자체가 정상 동작하지 않으나, 채팅방에 직접 진입하면 메시지 확인은 가능해 우회 경로는 존재함)
Frequency : Always (오픈채팅방 메시지 전송 후 검색 시 매번 결과에 노출되지 않는 것으로 확인됨)
Component : 오픈채팅
Phase : PRODUCTION
Environment : [iOS] iPhone 12 Pro Max (17.4) / [앱버전] 26.2.1

Description :
[사전조건] 오픈채팅방에 참여 중이며 글로벌 검색 기능을 사용할 수 있는 상태이다.
[재현스텝]
1. 오픈채팅방에 입장한다.
2. 메시지를 전송한다.
3. 글로벌 검색에서 전송한 메시지의 키워드로 검색한다.
[실행결과] 전송한 메시지가 검색 결과에 노출되지 않는다.
[기대결과] 오픈채팅방에서 전송한 메시지도 일반 채팅방 메시지와 동일하게 글로벌 검색 결과에 노출되어야 한다.
[참고사항] 오픈채팅 메시지만 검색 인덱싱 대상에서 제외되어 있는지 서버/인덱싱 로직 확인이 필요함.
